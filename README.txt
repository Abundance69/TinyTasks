TinyTasks AppKit v9.5

New:
- Option-click any selection circle to clear all currently selected circles.
- The list title is now left-aligned with a 12-point inset.

Existing v9.4 behavior remains:
- Click circles to toggle selection.
- Shift-click selects a visible range.
- Command-Shift-click adds a visible range.
- Drag selected circles to move tasks.
- Right-click circles for the task menu.
- Three-dot row buttons remain removed.

Run:
1. Quit Xcode.
2. Unzip this version into a fresh folder.
3. Open TinyTasks.xcodeproj.
4. Select TinyTasks > My Mac.
5. Press Command-R.

Existing tasks and appearance settings are preserved.
